# Serendib Gems ERP — Team Sync Point

A single self-contained HTML page that holds the full plan for the Serendib Gems ERP group project: overview and marks, glossary, the SRS, the implementation plan, the database design, the business process analysis, the git workflow, the stack decision, and how the modules connect.

It is one file, `index.html`, with all CSS and JavaScript inside it. No build step, no dependencies.

## View it live
Hosted through raw.githack.com (a raw GitHub CDN, no custom domain):

`https://raw.githack.com/agentdynarq/serendib-team-hub/main/index.html`

## Edit it
Open `index.html` in any editor. The content is plain HTML. Change text, run nothing, just commit and the live link updates.

## Notes
- Light theme by default, with a dark mode toggle (saved per browser).
- Works on phone, tablet and desktop.
- This is the reading hub. The actual code and documents live in the main project repo `serendib-erp`.
